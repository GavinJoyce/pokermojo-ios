# PokerMojo for iPad

A native SwiftUI poker hand comparison game for iPad.

**Compare 20 poker hands as fast as you can!**

## Features

- **Two Game Modes**: Standard (clear-cut differences) and Hard (tricky kickers & close calls)
- **44 Curated Scenarios**: Hard mode includes carefully crafted tricky hands
- **Realistic Card Design**: Corner indices with large center pip
- **Sound Effects**: Audio feedback for countdown, correct/wrong answers, and victory
- **Leaderboard**: Separate leaderboards for Standard and Hard modes
- **Review Mode**: See which hands you got wrong after each game

## Requirements

- iOS 17.0+
- iPad

## Installation

1. Clone this repository
2. Open `PokerMojo/PokerMojo.xcodeproj` in Xcode
3. Build and run on your iPad or simulator
